# FioreDiScatolaWeb

Modern, responsive florist website template built with React + Tailwind CSS.

## Requirements

- Node.js 18+ (recommended: latest LTS)
- npm (comes with Node.js)

## Run the webpage locally

1. Open a terminal in the project folder:

   ```bash
   cd /Users/manuel/Desktop/FioreDiScato/FioreDiScatolaWeb
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm run dev
   ```

4. Open the local URL shown in terminal (usually):

   [http://localhost:5173](http://localhost:5173)

## Build for production

Create an optimized production build:

```bash
npm run build
```

## Preview production build locally

After building, preview the production output:

```bash
npm run preview
```

Then open the URL shown in terminal (usually `http://localhost:4173`).

## Available scripts

- `npm run dev` - start development server
- `npm run build` - generate production build into `dist/`
- `npm run preview` - preview the production build locally
